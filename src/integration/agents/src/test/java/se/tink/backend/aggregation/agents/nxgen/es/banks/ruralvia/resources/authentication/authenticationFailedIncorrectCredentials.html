Error de acceso


Has introducido de forma incorrecta alguno de los datos de acceso.

  • Verifica que el NIF/NIE corresponde al usuario indicado (se admiten mayúsculas y minúsculas) y que has tecleado las letras y números seguidos (sin espacios ni guiones).
  • Verifica que la contraseña que has introducido se corresponde con el mismo formato (mayúsculas/minúsculas) que elegiste.

Recuerda que, por seguridad, tres errores consecutivos en la contraseña bloquearán el acceso. Si has olvidado tu contraseña pulsa recordar clave.

Si tras seguir estas instrucciones continúa el error, llama al 913 346 780 o acude a tu oficina.
